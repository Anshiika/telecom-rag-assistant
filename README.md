# Telecom RAG Chatbot

An AI-powered telecom assistant built using Retrieval-Augmented Generation (RAG).

## Features

- Document ingestion
- Text chunking
- Embedding generation
- Vector database
- Semantic retrieval
- LLM-powered responses
- Chat interface

## Tech Stack

- Python
- LangChain
- FAISS / ChromaDB
- Hugging Face
- OpenAI / Gemini
- Streamlit (later)
